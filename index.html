Selectors Syntax Description
Universal * {} Selects all elements in HTML.
Tags/Types type {} Selects all elements of with particular tag - <p>.
Classes .className {} Selects all elements of that class - class="className".
IDs #IdName {} Selects all HTML elements of with id="idName".
Attribues type[attributeName*="value"] Selects all HTML elements of type with particular attribute whose value contains the word 'value'.
Pseoudo Classes type: pseudoClass Selects all tags with pseudo class - a:active
Chaining Selectors .className1.ClassName2 Selects all elements with ClassName1 and ClassName2
Descendant Combinators .className type Selects all tags nested under element with className as class
Multiple Selectors type, class, id {} Apply ruleset to elements selected by multiply unrelated selectors.
